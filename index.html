🔄 Factory Reset — Reset to Original
Completely resets the entire application to a clean factory state. Deletes ALL data records AND the manager account AND all employee accounts AND all sessions. After reset, the app behaves as if opened for the very first time — manager account setup is required again. Use this to reset after testing or to hand over to a new team.
⚠ Danger Zone — Irreversible Actions
Hard Reset — Clear ALL Program Data Wipes all audits, Kaizen reports, corrective actions, saved reports, safety observations, and points. Employee accounts and manager account are kept. Cannot be undone. Export backup first.
Clear Audit Records Only Removes all 5S audit records. Kaizen reports and employee accounts preserved.
Clear Kaizen Reports Only Removes all Kaizen report records. Audit data preserved.
Clear Corrective Actions Removes all CAPA tracker entries.
Clear Safety Observations Removes all near-miss and safety observation records.
Reset Employee Points Resets all employee points to zero. Accounts and records preserved.
Clear File Library Permanently deletes all uploaded files (PDF, DOCX, XLSX, etc.) from the library. Cannot be undone.
💡Always export a full backup (Dashboard → Export Tracker XLSX) before performing any data reset operation.
